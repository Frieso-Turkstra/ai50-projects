Sentential Logic Validator


What is it?

The Sentential Logic Validator checks the validity of arguments from propositional logic. An argument is invalid if all its premises are true while its conclusion is false, otherwise the argument is valid. If the argument is invalid, the invalidating assignment will be marked red in the truth table.

For more information on sentential logic and validity, see this primer by Richard Grandy & Daniel Osherson: Sentential Logic Primer
